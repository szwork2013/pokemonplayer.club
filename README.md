# pokemonplayer.club


## Environment
- node : 6.3 <= node -v < 7
- npm : 3.10 <= npm -v < 4

## Running

### Development
```shell
$ cp config.js.example config.js
$ npm install
$ npm run start:dev
```

### Production
```shell
$ cp config.js.example config.js
$ npm install
$ npm run build
$ npm run start:prod
```
